working with git

1. add files to your project

this can be done via the touch command or through the interface as usual

examples $touch index.html && touch main.css

2. add new or modified files to staging

once we add new files or modify existing files we can tell git to track then by adding them to staging fo the next commit
think of commts as chekpoints. This will let git know what files the next checkpoint will consist of .

$git add

or

$git add .

your files are now on stage to be included in the next commit/checkpoint

3. add a commit and a commit message

with your changes staged, you can creat the checkpoint in your work with the following command.

$git commit -m "commit message"

this command creates the checkpoint fo git! pretty simple

4. push your code

finally push your changes up to github with the following command:

$ git push
